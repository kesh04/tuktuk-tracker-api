# Sri Lanka Police Three Wheeler Tracking System API

## Student ID: COBSCCOMP242P-044
##

## Overview
This RESTful API provides real time tracking capabilities for three wheelers (tuk-tuks) to assist Sri Lanka Police in monitoring and investigation operations.

## Technologies Used
- Node.js with Express.js
- MongoDB with Mongoose ODM
- JWT Authentication
- Swagger/OpenAPI Documentation
- bcrypt for password hashing
